# Xcode Practice

Use this repository in Xcode by opening `Package.swift`.

## Why This Exists

The repository still keeps standalone lesson files for reading and review, but direct hands-on practice now starts from multi-target Objective-C package targets.

Each target is meant to be opened, edited, built, and rerun inside Xcode.

## How To Open

1. open `Package.swift` in Xcode
2. wait for package indexing
3. choose one executable target
4. edit `.h` and `.m` files directly
5. run the target
