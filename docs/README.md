# Docs

This folder groups the planning and practice guidance for `PFOCSd`.

Use it like this:

- `STUDY_PATHS.md`: choose a route based on your goal
- `CHALLENGES.md`: pick one concrete change after reading example code
- `learning_paths/30_day_plan.md`: follow a day-by-day training loop

Suggested order:

1. read the root `README.md`
2. choose one route in `STUDY_PATHS.md`
3. practice a lesson or project
4. use `CHALLENGES.md` to extend what you just ran
