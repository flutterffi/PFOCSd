# Projects

Use this folder for hands-on practice after you finish a few lessons.

Current structure:

- `01_task_board_cli`: a small Objective-C CLI project that now supports loading, saving, filtering, and multiple render styles
- `02_architecture_comparison_lab`: a planned app-architecture comparison lab for MVC, MVVM, MVP, and VIPER

Current architecture comparison progress:

- MVC: runnable package target
- MVVM: runnable package target
- MVP: next target to add
- VIPER: next target to add

Best way to practice here:

1. run the current project
2. change one behavior
3. save or load data from a file
4. add one new option or one new model rule

Suggested next modifications:

- add a new state
- add tag filtering
- add sorting options
- add a compact report export
- build the same app in MVC, MVVM, MVP, and VIPER for architecture comparison
